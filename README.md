# life-game 🗡️🌿

ファンタジーライフ風の、ブラウザだけで遊べるブラウザゲームです。
**採集 → 装備作成 → 強化 → 戦闘** のループを楽しめます。

## 遊び方

ビルドやサーバー不要。`index.html` をブラウザで開くだけで遊べます。

```
# そのままダブルクリックで開くか、簡易サーバーで開く場合：
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

## ゲームの流れ

1. **⛏ 採集** — エリアの木・石・鉱脈から素材を集める。良い道具ほどたくさん採れる。
2. **🔨 作成** — 集めた素材で「斧・ピッケル（道具）」「武器」「防具」を作る。
   - 道具のランクを上げると、奥のエリアの硬い木や鉱脈を採れるようになる。
3. **🎽 装備** — 作った装備を身につけて ATK / DEF / 最大HP を強化。
4. **🗺 探索** — モンスターとターン制バトル。勝つと経験値・ゴールド・素材ドロップ。
5. レベルが上がると新しいエリア（森 → 岩山 → 火口）が解放され、より強い敵と希少な素材に挑める。

## 戦闘システム

武器の種類によって戦い方が変わります（＝装備を変えると戦術が変わる）。

| 武器タイプ | 特徴 |
|---|---|
| 🗡️ 剣 | バランス型・そこそこ会心 |
| 🪓 斧 | 高火力だが会心しない |
| 🔱 槍 | 敵の防御を半分貫通 |
| 🏹 弓 | 1ターンに2回攻撃 |
| 🪄 杖 | 防御無視の魔法・回復も可 |
| 🔪 短剣 | 高会心・会心で毒 |

- **MP / スキル** … 武器タイプ固有の必殺技。`防御`でMP回復＆被ダメ軽減。
- **会心（クリティカル）** … 一定確率でダメージ1.8倍。
- **状態異常** … 毒・火傷（継続ダメージ）、スタン（敵が行動不能）。
- **属性（🔥炎 / ❄️氷 / ⚡雷）** … モンスターの弱点を突くとダメージ1.5倍。属性武器を作り分けて攻略しよう。

## 特徴

- 進行状況はブラウザの `localStorage` に自動セーブ
- 4 エリア / 8 種のモンスター / 6 武器タイプ・3 属性 / 23 種のクラフトレシピ
- 外部ライブラリ不要・完全オフラインで動作
