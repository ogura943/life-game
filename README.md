# life-game 🗡️🌿

ファンタジーライフ風の、ブラウザだけで遊べるブラウザゲームです。
**採集 → 装備作成 → 強化 → 戦闘** のループを楽しめます。

## 遊び方

ビルドやサーバー不要。`index.html` をブラウザで開くだけで遊べます。

```
# そのままダブルクリックで開くか、簡易サーバーで開く場合：
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

## ゲームの流れ

1. **🗺 探索** — エリアを選び、**全10ステージの連戦（ガントレット）**にコマンド式ターンバトルで挑む。
   - **ステージ5＝⭐中ボス、ステージ10＝👑大ボス**（第2形態あり）。HP/MPは持ち越し、ステージ間に小回復。
   - 各ステージで経験値・ゴールド・**素材ドロップ**（素材はすべて敵から手に入る）。10ステージ制覇でクリアボーナス。
2. **🔨 作成** — ドロップ素材で「武器」「防具」「装飾品」を作る。
3. **🎽 装備 / 強化** — 装備で ATK / DEF / HP を強化。さらに +1〜+10 に強化できる。
4. レベルが上がると新エリア（森 → 岩山 → 火口）が解放。奥ほど強敵＆希少素材。

## 戦闘システム

武器の種類によって戦い方が変わります（＝装備を変えると戦術が変わる）。

| 武器タイプ | 特徴 |
|---|---|
| 🗡️ 剣 | バランス型・そこそこ会心 |
| 🪓 斧 | 高火力だが会心しない |
| 🔱 槍 | 敵の防御を半分貫通 |
| 🏹 弓 | 1ターンに2回攻撃 |
| 🪄 杖 | 防御無視の魔法・回復も可 |
| 🔪 短剣 | 高会心・会心で毒 |

- **MP / スキル / 道具** … 武器タイプ固有の必殺技、`防御`でMP回復＆被ダメ軽減、`道具`で回復アイテム使用。
- **会心（クリティカル）** … 一定確率でダメージ1.8倍。
- **状態異常** … 毒・火傷（継続ダメージ）、スタン（行動不能）。**敵にも自分にも**発生する。
- **属性（🔥炎 / ❄️氷 / ⚡雷）** … 弱点を突くと1.5倍、耐性には0.5倍。属性武器を作り分けて攻略しよう。
- **👑 ボス戦** … 各エリアにボスが出現。HPが減ると怒り（攻撃力UP）、逃走不可。撃破で大量の報酬と「ボスの魂」を入手し、最強装備の素材になる。
- **演出＆効果音** … ダメージ数字ポップ・被弾フラッシュ・シェイク、Web Audioで合成した効果音／BGM（設定でON/OFF）。
- **🔁 オート / 直前と再戦** … 戦闘をオート進行、探索画面から同じ敵にすぐ再挑戦。

## 🔨 装備と強化

- 武器・防具・**装飾品（アクセサリ）** をクラフト。アクセサリは会心率・状態異常耐性・ドロップ率などビルドの幅を広げる。
- **強化 +1〜+10** … ゴールドと魔石で装備を底上げ。`装備`タブから強化できる。

## 🛒 お店

戦闘で稼いだゴールドの使い道：

- **消耗品**（戦闘中に使用）… 回復ポーション / マナポーション / 万能薬（状態異常治療）/ エリクサー
- **永続強化の種** … ちから・まもり・いのちの種でステータスを恒久アップ（買うほど高くなる）
- **素材の売却** … 余った素材をゴールドに換金できる。

## 📖 記録（クエスト＆図鑑）

- **クエスト** … 「〇〇を倒す／集める」「ボス討伐」などを達成すると報酬がもらえる。
- **モンスター図鑑** … 倒したモンスターの絵・撃破数・弱点・ドロップを記録。

## 特徴

- 進行状況はブラウザの `localStorage` に自動セーブ
- 4 エリア＋4 ボス / 8 種のモンスター / 6 武器タイプ・3 属性 / 38 種のクラフトレシピ（武器・防具・道具・アクセサリ）
- ドット絵キャラ（PNGを内蔵）／ PWA対応でスマホのホーム画面に追加してアプリのように起動
- 外部ライブラリ不要・完全オフラインで動作
